# 🔬 Laboratorios de Física 3D

Plataforma educativa de simulaciones físicas interactivas desarrolladas con **p5.js** en entornos 3D WebGL. El proyecto combina **métodos numéricos computacionales** con el motor de física profesional **JoltPhysics** para ofrecer experiencias de aprendizaje variadas y de alto rendimiento.

## 🎯 Objetivo

Proporcionar herramientas interactivas para la enseñanza y comprensión de fenómenos físicos mediante:
- Implementación de algoritmos de integración numérica desde cero
- Uso de motores de física industriales (JoltPhysics compilado a WebAssembly)
- Visualización en tiempo real en navegador sin instalaciones

## 🧮 Enfoques Metodológicos

### 1. Métodos Numéricos (Integradores Propios)
Implementaciones en JavaScript puro para comprender los fundamentos matemáticos:
- Estabilidad y precisión de diferentes esquemas
- Error acumulado y conservación de energía
- Comparación visual entre métodos

### 2. Motor de Física JoltPhysics
Integración del motor C++ compilado a WebAssembly mediante Emscripten:
- Simulaciones de alto rendimiento con colisiones complejas
- Sólidos rígidos, restricciones y joints
- Detección de colisiones broad-phase y narrow-phase optimizadas
